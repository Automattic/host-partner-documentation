# Fetch Activity for a Site

### Endpoint Information

- __Method__: GET
- __URL__: `https://public-api.wordpress.com/wpcom/v2/sites/<blog_id>/activity`
- __blog_id__: Site to request activities for.

### Request Parameters

- __action__: Action to filter by.
- __after__: Return activities after this time.
- __before__
- __by__
- __date_range__
- __group__
- __locale__
- __name__
- __not_group__
- __number__
- __on__
- __page__
- __search_after__
- __sort_order__
- __aggregate__
- __aggregate_show__
